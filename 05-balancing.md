# Balancing — City Baron

---

## 1. ECONOMIC BALANCE (Zahlenbasis)

### Startkapital
- **Spieler:** 100.000 €
- **Bots:** 100.000 € (identisch)
- **Kreditlimit Start:** 50.000 €

### Kostenverhältnisse
| Größe | Verhältnis zu Startkapital |
|-------|--------------------------|
| Günstigste Wohnung | ~5 % (5.000 €) |
| Teuerste Immobilie (Einkaufszentrum) | ~300 % (300.000 €) |
| Durchschnittliche Immobilie | ~20–40 % (20.000–40.000 €) |
| Monatliche Miete (günstig) | 0,2 % des Kaufpreises |
| Monatliche Unterhaltskosten | 0,1–0,5 % des Kaufpreises |

### Marktgröße
| Gesamtmarkt (alle Bezirke) | 12–18 Bezirke × 10–20 Grundstücke |
|---------------------------|-----------------------------------|
| **Summe Grundstücke** | 150–300 pro Spiel |
| **Gesamtmarktwert (Start)** | ~3–8 Mio. € |

## 2. ZEITLICHES BALANCING

### Rundensystem
| Aspekt | Wert |
|--------|------|
| Runden für ein volles Spiel | 50–100 |
| Durchschnittliche Spieldauer | 20–45 Min. |
| Aktionen pro Runde (Spieler) | 1–3 |
| Zeit pro Runde (Planung) | 30–60 Sek. |
| Runden bis erste merkliche Expansion | 5–10 |
| Runden bis Midgame | 15–25 |

### Wachstumsgeschwindigkeit
| Strategie | Marktanteil nach 30 Runden |
|-----------|---------------------------|
| Aggressiv | 20–35 % |
| Defensiv | 10–20 % |
| Spekulativ | 15–30 % |
| Ausgeglichen | 15–25 % |

## 4. COMEBACK-BALANCING

| Mechanik | Wirkung | Auslöser |
|----------|---------|----------|
| **Catch-up Bonus** | +20 % Mieteinnahmen | Marktanteil < 10 % |
| **Gläubigerschutz** | 2 Runden Schonfrist | Kapital ≤ 0 |
| **Investor-Notkredit** | 10.000 € zu 15 % Zinsen | Einmalig bei Insolvenz |
| **Zwangsversteigerung** | 60 % Marktwert Startpreis | Nach Gläubigerschutz |
| **Schonfrist-Kaufoption** | Eine Wohnung bleibt erhalten | Bei Zwangsversteigerung |

---

## 5. UNTERNEHMENSGRÖSSEN-EFFEKTE

### Vorteile großer Spieler:
- Synergie-Boni (Quartiersbonus)
- Niedrigere Kreditzinsen (bessere Bonität)
- Größeres Kreditlimit

### Nachteile großer Spieler:
- Höhere absolute Unterhaltskosten
- Mehr Angriffsfläche für Konkurrenz
- Schwerer zu verwalten (mehr Immobilien)
- Preisdumping-Angriffe treffen härter
- Verkauf großer Portfolios senkt Marktpreise

---

## 6. BOT-SCHWIERIGKEIT

| Schwierigkeit | Bot-Fehlerrate | Reaktionszeit |
|--------------|---------------|---------------|
| Leicht | 20–30 % Fehler | 3 Runden Verzögerung |
| Normal | 5–10 % Fehler | 1 Runde Verzögerung |
| Schwer | 0–5 % Fehler | 0 Verzögerung |

Bots erhalten **nie** Ressourcen-Boni (kein Cheating).

---

## 7. MARKT-REGULATION

- Maximal 70 % Marktanteil für einen Spieler, bevor Gegenreaktionen einsetzen
- Ab 50 % Marktanteil: Kartellamt-Prüfung (Zufalls-Event)
- Ab 60 % Marktanteil: Andere Spieler erhalten Koalitions-Boni gegen den Marktführer
